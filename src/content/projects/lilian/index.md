主流健身 App（如 Keep）只提供固定模板，无法支持深蹲 / 卧推 / 硬拉的科学周期化（储备期 / 转化期 / 减载）；而专业运动员用的软件门槛又太高。力练面向普通健身人群，填补这块空白。

- **核心功能**：根据用户 1RM 自动生成 28 周训练计划（储备期 → 转化期线性渐进），训练中实时记录完成情况，并根据完成率与疲劳度智能调整下次计划。
- **技术方案**：React Native + Expo Router。
- **AI 驱动开发**：使用 Claude Code + Superpowers 工程化工作流驱动全程开发，**单人 3 天**完成从需求到 demo。
- **成果**：发布抖音 demo 视频，**单日涨粉 150+**，评论区收到真实用户反馈，验证了产品方向。
