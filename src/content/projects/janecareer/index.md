《简职》是一款 AI 驱动的智能求职系统，致力于解决求职中重复劳动多、反馈机制缺失的痛点 —— 通过一键自动填写申请、智能职位推荐等功能提升求职效率。

- **智能表单自动填写**：基于 LLM 理解页面结构与字段语义，无需单站手动适配，跨平台通用。
- **简历解析与优化**：结合岗位要求做关键词匹配与结构优化建议，提升通过率。
- **个性化职位推荐**：基于用户画像与行为数据做精准匹配。
- **我的贡献**：独立负责插件前端框架从 0 到 1 的搭建、UI 组件选型与核心页面逻辑；完成简历智能填写 MVP 原型；并指出与设计跨平台表单 JSON 适配方案，为团队攻克核心难点提供方向。
